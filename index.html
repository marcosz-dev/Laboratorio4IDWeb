Pasos para un algoritmo que considere condiciones y repeticiones (Ejemplo)

  1. Inicio
  2. Entrada de datos: se reciba una lista de numeros
  3. Inicializacion: se inicializa una variable max_numero con el primer valor de la lista
  4. Bucle: se recorre la lista de numeros
  5. Condicion: Cada numero en la lista es comparado con max_numero
  6. Actualizacion: Si el numero actual es mayor que max_numero, se actualiza max_numero con el valor del numero actual
  7. Salida de datos: Se muestra el valor de max_numero
  8. Fin
